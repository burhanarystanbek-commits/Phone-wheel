# PhoneWheel v2.0

Превращает Android-телефон в руль, педали и кнопки для гоночных симуляторов.
Телефон соединяется с ПК по **Wi-Fi**, **USB** или **Bluetooth**, и Windows видит его как обычный джойстик через vJoy.

---

## Что нужно

| | |
|---|---|
| **Телефон** | Android 7.0+ (API 24+) |
| **ПК** | Windows 10/11, 64-bit |
| **Игры** | Любая игра с поддержкой джойстика — Assetto Corsa, F1, iRacing, BeamNG и др. |

---

## Быстрый старт

### 1. Установи vJoy (один раз)

Скачай с [sourceforge.net/projects/vjoy](https://sourceforge.net/projects/vjoy/) и установи.

После установки открой **Configure vJoy** и убедись что устройство **#1** включено
с осями **X** (руль), **Y** (газ), **Z** (тормоз) и хотя бы **8 кнопками**.

> Либо запусти PhoneWheel.exe — кнопка **«Настроить vJoy»** откроется автоматически
> если что-то не так, и покажет что именно.

### 2. Запусти PhoneWheel на ПК

Скачай `PhoneWheelPC.exe` из последнего [Release](../../releases) или из
[Actions → Build PC App (exe)](../../actions/workflows/build-pc-app.yml) → артефакт `PhoneWheelPC-win64`.

Запусти. В первый раз Windows может попросить разрешить доступ к сети — разреши.

### 3. Установи APK на телефон

Скачай APK из [Actions → Build APK](../../actions/workflows/main.yml) → артефакт.

Перед установкой разреши «Установку из неизвестных источников» в настройках Android.

## Режимы подключения

### Wi-Fi

1. Телефон и ПК должны быть в одной Wi-Fi сети.
2. На ПК: режим **Wi-Fi** (по умолчанию).
3. На телефоне: вкладка **Wi-Fi** → введи IP-адрес ПК → **Подключить**.

Узнать IP на Windows: `Win + R` → `cmd` → `ipconfig` → строка «IPv4-адрес».

### USB

1. Установи [ADB (Android Debug Bridge)](https://developer.android.com/tools/releases/platform-tools).
2. Подключи телефон кабелем, разреши отладку по USB на телефоне.
3. На ПК: нажми **«Настроить USB (adb reverse)»** — это пробросит порт через кабель.
4. На телефоне: вкладка **USB** → **Подключить**.

Работает даже без Wi-Fi, задержка обычно ниже.

### Bluetooth

**На ПК:**
1. Убедись что Bluetooth включён в настройках Windows.
2. Сопряги телефон с ПК через *Настройки → Bluetooth и устройства → Добавить устройство*.
3. Выбери режим **Bluetooth** в PhoneWheel.exe — приложение начнёт ожидать подключение.

**На телефоне:**
1. Выбери вкладку **BT**.
2. Нажми **«Выбрать»** → выбери свой ПК из списка сопряжённых устройств.
3. Нажми **«Подключить»**.

> Если телефон не видит ПК — нажми **«Искать ещё»** в диалоге выбора устройства.
> Впервые потребуется разрешить Bluetooth-разрешения в Android.

## Кнопки

### Добавить кнопку на телефоне

1. Нажми ⚙ (иконка редактирования) в углу экрана.
2. Нажми **«+ Добавить кнопку»** — появится новая кнопка по центру.
3. **Перетащи** за тело, **потяни за уголок** (фиолетовая точка) чтобы изменить размер.
4. **Тап** по кнопке → переименовать.
5. **Долгое нажатие** → удалить.
6. Нажми **«Готово»**.

Кнопки сохраняются автоматически и восстанавливаются при следующем запуске.

### Назначить кнопку в PhoneWheel.exe

Когда кнопка на телефоне нажата хотя бы раз, она появится в таблице на ПК автоматически.

Для каждой строки задай:
- **Название** — для удобства (не влияет на игру)
- **vJoy #** — номер кнопки виртуального джойстика (1–32)

Нажми **«Сохранить маппинг»**.

### Назначить кнопку в игре

В настройках управления игры найди **Steering Wheel** или **Joystick**, выбери нужное действие и нажми нужную кнопку на телефоне. Игра увидит её как **vJoy Button N**.

## Настройка руля

| Настройка | Где |
|-----------|-----|
| Угол поворота | Ползунок в приложении на телефоне |
| Центрование | Нажми **«Центр»** когда держишь телефон в нейтральном положении |

Телефон держи **горизонтально** (landscape) как настоящий руль.

## Dashboard (ПК)

В главном окне PhoneWheel.exe показывается:

| Блок | Что показывает |
|------|----------------|
| **ПОДКЛЮЧЕНИЕ** | Режим, имя устройства, время соединения |
| **КОНТРОЛЛЕР** | Текущие значения руля/газа/тормоза |
| **ПИНГ / ПАКЕТЫ** | Задержка (цветная точка) и частота пакетов |
| **vJOY** | Статус драйвера |

**Цветовая шкала пинга:**
🟢 < 20 мс — отлично  
🟡 < 50 мс — хорошо  
🟠 < 100 мс — приемлемо  
🔴 > 100 мс — высокая задержка

## Возможные проблемы

**«vJoy не найден» / «Устройство недоступно»**
→ Нажми **«Настроить vJoy»** в приложении. Следуй инструкциям диалога.

**Телефон не подключается по Wi-Fi**
→ Проверь что оба устройства в одной сети. Отключи VPN. Проверь что антивирус/брандмауэр не блокирует порт 27111.

**«Нет доступа к порту» (USB/Wi-Fi)**
→ Нажми **«Настроить доступ»** — это зарегистрирует URL-резервирование для порта 27111. Нужно один раз, требует прав администратора.

**Bluetooth: телефон не находит ПК**
→ Убедись что ПК сопряжён с телефоном через *Настройки Windows → Bluetooth*. Bluetooth-сопряжение нужно сделать один раз через системные настройки до того как подключаться через PhoneWheel.

**Руль дрейфует / не центрируется**
→ Нажми **«Центр»** в приложении, держа телефон ровно горизонтально.

## Технические детали

- Протокол: JSON по строкам (newline-delimited), одна `WheelState` в строке
- Частота: 100 Гц (пакет каждые 10 мс)
- Keepalive: heartbeat каждые 500 мс
- Watchdog: авторестарт если 3 сек нет пакетов
- Порт WebSocket (Wi-Fi / USB): `27111`
- Bluetooth UUID: `8ce255c0-200a-11e0-ac64-0800200c9a66`

## Сборка из исходников

```
# Требуется JDK 17+
./gradlew assembleDebug
# APK: app/build/outputs/apk/debug/app-debug.apk
```

### Windows EXE

```
# Требуется .NET SDK 8.0+
dotnet publish pc-app/PhoneWheelPC.csproj \
  -c Release -r win-x64 --self-contained true \
  -p:PublishSingleFile=true \
  -o publish
# EXE: publish/PhoneWheelPC.exe
```

Оба артефакта также собираются автоматически через GitHub Actions при каждом push.
