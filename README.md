# PhoneWheel

Превращает телефон в руль с педалями для PC-гонок (Assetto Corsa, F1, iRacing
и другие — любая игра, которая умеет работать с обычным контроллером).

Состоит из двух частей:

- **Android-приложение** (`app/`) — руль (наклон телефона), газ/тормоз
  (свайп по экрану) и набор кастомных кнопок (передачи, ручник, DRS,
  pit limiter), которые можно свободно перетаскивать и менять размер.
  Шлёт состояние по WebSocket 60 раз в секунду.
- **PC-приложение** (`pc-app/`) — принимает данные и выводит их в
  виртуальный джойстик через **vJoy**, чтобы игры видели обычный контроллер.
  Подробности и установка: [`pc-app/README.md`](pc-app/README.md).

## Подключение

- **Wi-Fi**: телефон и PC в одной сети → в приложении на телефоне ввести IP
  PC.
- **USB**: включить USB-отладку на телефоне, на PC нажать "Настроить USB" в
  PC-приложении (использует `adb reverse`).

## Кастомизация кнопок на телефоне

Нажми на шестерёнку (⚙) в левом верхнем углу — откроется панель редактирования:

- **Шаблон** — готовая раскладка под Assetto Corsa / F1 / iRacing.
- **+ Кнопка** — добавить ещё одну (передача вверх/вниз, ручник, DRS,
  pit limiter).
- Перетаскивай кнопки пальцем, тяни за нижний правый уголок, чтобы изменить
  размер. Долгое нажатие — удалить кнопку.
- **Готово** — выйти из режима редактирования. Раскладка сохраняется
  автоматически и восстанавливается при следующем запуске.

Какую игровую кнопку (vJoy) триггерит каждое действие — настраивается **в
PC-приложении**, а не на телефоне: положение кнопки на экране телефона не
влияет на то, что происходит в игре.

## Протокол (WebSocket, JSON, ~60 Hz)

```json
{
  "type": "state",
  "seq": 1234,
  "steer": -0.42,
  "throttle": 0.81,
  "brake": 0.0,
  "buttons": { "gear_up": false, "gear_down": true, "handbrake": false, "drs": false, "pit_limiter": false },
  "preset": "assetto_corsa"
}
```

## Сборки

Готовые `.apk` (Android) и `.exe` (PC, Windows) собираются автоматически в
GitHub Actions при каждом пуше — смотри вкладку **Actions** репозитория,
артефакты `PhoneWheel-debug` и `PhoneWheelPC-win64`.
